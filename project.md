# Coffee Shop Sales Analysis Dashboard

## Project Overview
This project focuses on analyzing coffee shop sales data to identify business trends, customer behavior, and product performance using data analysis and visualization techniques.

The dashboard helps understand sales patterns, peak business hours, top-selling products, and revenue performance to support better business decisions.

## Objectives
- Analyze overall sales performance
- Identify peak sales hours and busy days
- Find top-selling and low-performing products
- Understand customer purchasing patterns
- Visualize business insights through interactive dashboards

## Tools Used
- Microsoft Excel
- Power BI
- Data Cleaning
- Data Visualization
- Dashboard Design

##  Dataset Information
The dataset contains coffee shop transaction records including:
- Transaction ID
- Date and Time
- Product Category
- Product Name
- Quantity Sold
- Unit Price
- Sales Amount
- Store Location

##  Dashboard Features
- Total Sales KPI
- Total Orders
- Revenue Trends
- Product Category Analysis
- Top-Selling Products
- Hourly Sales Analysis
- Daily and Monthly Performance
- Store Location Comparison

##  Key Insights
- Peak sales occur during specific business hours
- Certain products generate higher revenue
- Sales patterns vary across locations and time periods
- Customer demand trends help improve inventory and marketing decisions

## Project Outcome
This dashboard transforms raw sales data into meaningful insights that help businesses make data-driven decisions and improve operational performance.
